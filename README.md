# CMS – Campus Management System

A beginner ASP.NET Core MVC project to manage Employee data using SQL Server.

## Tech Stack
- ASP.NET Core MVC
- C# & Razor Views
- SQL Server (LocalDB)
- Bootstrap 5

## Features
- Add, View, Edit and Delete Employees
- Form validation
- Responsive UI with Bootstrap

## Database Tables
- `tbl_emp` — Employee (ID, Name, Dept, Salary)

## How to Run
1. Clone the repo
2. Open `CMS.sln` in Visual Studio
3. Install NuGet package: `System.Data.SqlClient`
4. Update the connection string in `EmployeeModel.cs`
5. Create the database tables and press `F5` to run
